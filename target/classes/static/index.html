What's Inside

Everything You Need

A complete, production-ready secure file storage platform — built from scratch in Spring Boot.

🔐

AES-128 File Encryption

Every file is encrypted using Advanced Encryption Standard (AES-128) before being stored on disk. Your data is unreadable without the key.

👤

User Accounts & Auth

Full registration and login system powered by Spring Security. Passwords are hashed using BCrypt — never stored in plain text.

📁

Private File Vault

Each user can only access their own uploaded files. Complete data isolation enforced at the service and API layer.

⭐

Admin Control Panel

Admin users get full visibility — view all registered users, all uploaded files, and system overview stats in a premium dashboard.

⚡

REST API Ready

Clean REST endpoints for upload and download. CSRF protection, role-based access, and proper HTTP status codes throughout.

🎨

Premium Dark UI

Glassmorphism design with animated orbs, particle effects, drag-and-drop upload zones, and smooth micro-animations across all pages.

About the Project

Built with passion & precision.

CryptoVault was created as a full-stack demonstration of how modern Java backend technology can be combined with elegant frontend design to produce a genuinely secure and visually stunning application.

The project uses Spring Boot 3 for the backend, Spring Security for authentication, Spring Data JPA for persistence, and Thymeleaf for server-side rendering — all wired together with a custom AES encryption utility class.

Every design decision — from the per-user file isolation to the admin role separation — was made to reflect real-world security principles in a beginner-friendly, clean codebase.
